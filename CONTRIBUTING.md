# Contributing to Predictive Supply Chain Optimizer

Thank you for your interest in contributing!

## How to Contribute
1. Fork the repo and create your branch from `main`.
2. Ensure your code follows the project's style (run linter).
3. Write unit tests for new features.
4. Open a Pull Request with a clear description of changes.

## Development Setup
- Use `docker-compose up` to run the stack.
- Run tests with `pytest`.
- Follow the Repository-Service pattern for database operations.
